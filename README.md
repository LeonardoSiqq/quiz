---
Funcionalidades
- Perguntas de conhecimentos gerais  
- Sistema de pontuação automática  
- Navegação simples entre perguntas  
- Interface intuitiva e leve  
- Execução direta no navegador (sem backend)
---
Tecnologias Utilizadas
- **HTML5** – Estrutura semântica da aplicação  
- **CSS3** – Estilização e layout  
- **JavaScript (Vanilla)** – Lógica do quiz e manipulação do DOM  
---
Estrutura do Projeto
quiz/  
├── index.html – Estrutura principal do quiz  
├── styles.css – Estilos visuais  
└── script.js – Lógica do quiz (perguntas, respostas e pontuação)

---

Como Executar Localmente

1. Clone o repositório:
git clone https://github.com/leonardosiqq/quiz.git

2. Acesse a pasta do projeto:
cd quiz

3. Abra o arquivo `index.html` no navegador.
